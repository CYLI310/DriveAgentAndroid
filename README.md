# DriveAgent

DriveAgent is an Android application that displays a map with speed trap locations. The application uses a WebView to render a map from a local HTML file and leverages a JavaScript interface to interact with native Android code for location updates and data retrieval.

## Features

- Displays a map using a WebView.
- Marks speed trap locations on the map from a GeoJSON file.
- Real-time location updates.

## How to Build and Run

1. **Clone the repository.**
2. **Open the project in Android Studio.**
3. **Build the project:**
   - Go to `Build` > `Make Project` or click the "Make Project" button in the toolbar.
4. **Run the application:**
   - Go to `Run` > `Run 'app'` or click the "Run 'app'" button in the toolbar.
   - Select an emulator or a connected physical device.
